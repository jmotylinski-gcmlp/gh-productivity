# GitHub Productivity Tracker

A tool to analyze and visualize daily lines of code written by a GitHub user. Fetches commit data from GitHub, calculates daily line changes, and provides both data exports and a web dashboard for visualization.

## Features

- Fetches commit data from GitHub API
- Calculates daily lines of code added/removed
- Exports data to JSON and CSV formats
- Web dashboard for visualization and analysis

## Getting Started

### Prerequisites

- Python 3.8+
- GitHub Personal Access Token (for API access)

### Installation

```bash
pip install -r requirements.txt
```

### Configuration

Create a `.env` file in the project root:

```
GITHUB_TOKEN=your_github_token_here
GITHUB_USERNAME=jasonmotylinski
```
